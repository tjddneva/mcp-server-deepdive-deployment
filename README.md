## Installation

To install `mcp-server-deepdive-deployments`, use the following command:

```json
"mcp-server-deepdive-deployments": {
    "command": "uvx",
    "args": [
        "--from",
        "git+https://github.com/tjddneva/mcp-server-deepdive-deployment.git",
        "mcp-server"
    ]
}
```

This configuration will fetch and deploy the `mcp-server` from the specified GitHub repository using `uvx`.
